Handwritten Digit Recognition Using CNN Overview: This project implements a Handwritten Digit Recognition System using a Convolutional Neural Network (CNN). The model is trained on the MNIST dataset, which contains thousands of handwritten digit images (0–9). The system can accurately classify handwritten digits from both the test dataset and custom user-provided images.

Features: Recognizes handwritten digits from 0 to 9 Uses Convolutional Neural Network (CNN) for classification Trained on the MNIST dataset Supports prediction on custom handwritten images Achieves high accuracy (approximately 98–99%) Simple and easy-to-understand implementation

Technologies Used: Python TensorFlow / Keras NumPy OpenCV Matplotlib Dataset

The project uses the MNIST Dataset, a benchmark dataset containing: 60,000 training images 10,000 testing images 28 × 28 pixel grayscale images Digits ranging from 0 to 9 Project Workflow Load MNIST Dataset Preprocess Image Data Normalize Pixel Values Build CNN Model Train the Model Evaluate Model Performance Save Trained Model Predict Digits from Images Display Results
